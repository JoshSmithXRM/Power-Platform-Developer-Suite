# Release v0.2.2 - Data Explorer MVP & E2E Testing

**Release Date:** 2025-11-26
**Version:** 0.2.2
**Type:** Patch Release

---

## Overview

This release introduces the **Data Explorer** panel for executing SQL queries against Dataverse, along with comprehensive **E2E testing infrastructure** using Playwright.

---

## Highlights

### Data Explorer Panel (MVP)

Query your Dataverse data using familiar SQL syntax:

- **SQL to FetchXML** - Write SQL, see the equivalent FetchXML in real-time
- **Results Table** - Sortable columns with proper data formatting
- **Record Links** - Click any lookup field or primary key to open the record in Dataverse
- **Copy URL** - Hover over record links to copy the URL to clipboard
- **Syntax Highlighting** - SQL editor and FetchXML preview with proper highlighting
- **Keyboard Shortcut** - Press Ctrl+Enter to execute queries

### E2E Testing Infrastructure

Automated testing for the VS Code extension:

- **Smoke Tests** (`npm run e2e:smoke`) - Quick validation (~30s)
- **Integration Tests** (`npm run e2e:integration`) - Real Dataverse connections (~60s)
- **Screenshot Capture** - Automatic screenshots for debugging
- **Log Capture** - Extension and console logs available in test output

## Changes

### Added

- **Data Explorer Panel (MVP):** SQL query execution against Dataverse
  - SQL to FetchXML transpilation with live preview
  - Query results displayed in sortable table
  - Clickable record links - lookup fields and primary keys open records in browser
  - Copy record URL button on hover
  - Syntax highlighting for SQL editor and FetchXML preview
  - Query execution status bar with row count and timing
  - Keyboard shortcut (Ctrl+Enter) for query execution

- **E2E Testing Infrastructure:** Playwright-based E2E testing for VS Code extension
  - `npm run e2e:smoke` - Automated smoke tests
  - `npm run e2e:integration` - Integration tests with real Dataverse connections
  - VSCodeLauncher, CommandPaletteHelper, WebviewHelper utilities
  - ClaudeJsonReporter for AI-optimized test output
  - Data Explorer, Environment Setup, and Solutions Panel integration tests

### Changed

- **Dependencies:** Upgraded TypeScript to 5.9.3, @azure/msal-node to 3.8.3
- **CI:** Updated GitHub Actions to latest versions

---

## Upgrade Notes

No breaking changes. Simply update to enjoy the new Data Explorer panel.

To use Data Explorer:
1. Open Command Palette (Ctrl+Shift+P)
2. Run "Power Platform Developer Suite: Data Explorer"
3. Select an environment
4. Enter your SQL query and click "Execute Query"
